Welcome to Quiz Gen WEB



   Quiz Gen is a quiz app where you can create and attempt quizzes. Quiz Gen was first developed as a Python applicaition which had its own downsides like unavailability to be hosted online and make public. This web version will overcome that hurdle.

Some of the great features in Quiz Gen are:
 -- Generating quizzes seemlessly
 -- Quiz code format to attempt quizzes
 -- Discord server to backup questions and quiz attempt results
 -- Json question local file backup
 -- Dashboard to keep track of all quizzes attempted and created
